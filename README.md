# blog-app-vue

###Description
```
A simple web application where I can post my blogs, artworks and projects.
```
## Tech used
```
> Vue.js 3
> Vue router
> Vuex 4
> Bootstrap 5
> Cloudinary (Image storage)
> Laravel 8 (Backend)

```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
